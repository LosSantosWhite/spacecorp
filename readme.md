Запуск проекта:


* Создать тестовые значения для БД(slite3) и внести фикстуры

        docker-compose run filldb

* запустить сервер и 

        docker-compose up web

Запуск тестов

    docker-compose run test

Перейти по адресу:

    http://localhost:8000
